

ASP .NET Core
    É um Framework para criação de app web
    Usa o sistema de templates @razor engine
    
Padrão MVC
    Estrutura: View, Controllers, Models e também ViewModels

Entity Framework (ORM ou Object Relational Mapping é uma técnica de mapeamento objeto
 relacional que visa criar uma camada de mapeamento entre nosso modelo de objetos 
 (aplicação) e nosso modelo relacional (banco de dados) de forma a abstrair o acesso ao mesmo.)
 
DTO = Data Transfer Object
